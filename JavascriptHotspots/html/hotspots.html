This app demonstrates hotspots. Any mouse click event should travel down to the element below the grid

Press the keys Ctrl+Shift+Y to see the hotspot grid. The Colours go from purple to red -> purple less activity, red -> loads

Presssing the keys Ctrl+Shift+E will toggle the opacity of the grid when visible.

Presssing the keys Ctrl+Shift+A will delete the grid.
